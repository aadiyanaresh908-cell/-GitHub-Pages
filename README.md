# Static Website Deployment using GitHub Pages

Objective
Host a simple static website using GitHub Pages for free.

##  Tools Used
- Git
- GitHub
- HTML
- CSS


##  Deployment Steps
1. Created a repository named `username.github.io`
2. Added static files (HTML, CSS, JS)
3. Committed and pushed code to GitHub
4. Enabled GitHub Pages from Settings
5. Website live at: https://username.github.io
